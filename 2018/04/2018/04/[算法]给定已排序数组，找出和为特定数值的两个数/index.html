给定已排序数组，找出和为特定数值的两个数

Posted on 2018-04-08 Edited on 2018-07-01 In 算法

要求

给定已排序数组，找出和为特定数值的两个数。假定数组中的数值都不相同。

方法一：穷举法

T(n) = (n - 1 + 1) * n / 2 = (n^2) / 2 = O(n^2)，即时间复杂度：O(n^2)。

方法二

T(n) <= n - 1 = O(n) 时间复杂度：O(n)。
